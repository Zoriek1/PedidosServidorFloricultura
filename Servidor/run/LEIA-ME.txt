│   └─ Para o servidor completamente
│

│   └─ Configurações (porta, host, etc.)
│
├── 📦 requirements.txt
│   └─ Dependências Python (instale com: pip install -r requirements.txt)
│
├── 🐍 main.py
│   └─ Código principal (não mexa aqui, use os .bat!)
│
├── 📁 app/
│   └─ Código da aplicação Flask
│
└── 📁 templates/
    └─ Templates HTML


═══════════════════════════════════════════════════════════════
🚀 INÍCIO RÁPIDO
═══════════════════════════════════════════════════════════════

1️⃣ PRIMEIRA VEZ? Instale as dependências:
   Abra o PowerShell aqui e execute:
   
   pip install -r requirements.txt

2️⃣ INICIAR O SERVIDOR:
   
   Opção A - Com janela (recomendado para ver erros):
   Duplo clique em: INICIAR_AQUI.bat
   
   Opção B - Em background (sem janela):
   Duplo clique em: INICIAR_SERVIDOR_BACKGROUND.vbs

3️⃣ ACESSAR O PAINEL:
   Abra o navegador e acesse:
   
   http://localhost:5000

4️⃣ PARAR O SERVIDOR:
   Duplo clique em: PARAR_SERVIDOR.bat


═══════════════════════════════════════════════════════════════
⚙️ CONFIGURAR
═══════════════════════════════════════════════════════════════

Edite o arquivo "config.json" para alterar:

• Porta do servidor (padrão: 5000)
• Host (padrão: 0.0.0.0 = aceita conexões de qualquer IP)
• Debug mode (padrão: false)
• Broadcast (descoberta de rede para clientes)

═══════════════════════════════════════════════════════════════
❓ PROBLEMAS?
═══════════════════════════════════════════════════════════════

1. Servidor não inicia?
   → Use INICIAR_AQUI.bat para ver os erros
   → Verifique se o Python está instalado: python --version
   → Instale as dependências: pip install -r requirements.txt

2. Erro "Port 5000 already in use"?
   → Execute PARAR_SERVIDOR.bat
   → Aguarde 5 segundos
   → Tente novamente

3. Não consegue acessar de outro computador?
   → Verifique o firewall do Windows
   → Confirme que config.json tem "host": "0.0.0.0"

4. Mais ajuda?
   → Consulte ../docs/ para documentação completa
   → Veja ../README.md
